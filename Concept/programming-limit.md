제어 프로그램으로 무엇까지 만들 수 있을까
컴퓨터에서 일어나는 일을 추적하고 제어하는 프로그램은 가능한가

개발을 하다 보면 이런 궁금증이 생긴다.
프로그램으로 컴퓨터에서 일어나는 일을 어디까지 알 수 있을까
그리고 그 일들을 직접 제어하는 것도 가능할까

결론부터 말하면 가능하다.
다만 무엇을 어디까지 제어할 수 있는지는
운영체제와 권한 그리고 언어의 역할에 따라 달라진다.

제어 프로그램이란 무엇인가

제어 프로그램은 단순히 계산을 하는 프로그램이 아니다.
컴퓨터 안에서 일어나는 상태를 감지하고
그 상태에 따라 행동을 결정하거나 개입하는 프로그램이다.

예를 들면 이런 것들이다.

어떤 프로세스가 실행 중인지 확인한다
파일이 생성되거나 삭제되는 것을 감지한다
CPU나 메모리 사용량을 모니터링한다
특정 조건이 되면 작업을 중단하거나 실행한다

이런 프로그램은 실제로 많이 사용되고 있다.

컴퓨터에서 실제로 일어나는 일들

컴퓨터에서는 항상 여러 가지 일이 동시에 일어난다.

프로세스가 생성되고 종료된다
파일이 읽히고 쓰인다
네트워크 요청이 오고 간다
메모리 사용량이 변한다
CPU 사용률이 올라가거나 내려간다

운영체제는 이 모든 상태를 관리하고 있고
프로그래밍 언어는 운영체제가 제공하는 정보를 통해
이 상태를 조회하거나 일부를 제어할 수 있다.

자바나 파이썬으로 가능한 범위

자바와 파이썬은 모두 운영체제 위에서 동작하는 언어다.
그래서 직접 하드웨어를 제어하지는 못하지만
운영체제가 허용한 범위 안에서는 충분히 많은 일을 할 수 있다.

대표적인 예를 보자.

프로세스 상태 추적 프로그램

현재 실행 중인 프로그램 목록을 확인하는 것은 가능하다.

파이썬 예시

import psutil

for process in psutil.process_iter():
    print(process.pid, process.name())


이 코드는 현재 실행 중인 프로세스의 아이디와 이름을 출력한다.
작업 관리자와 비슷한 정보를 코드로 얻는 것이다.

자바에서도 비슷한 작업이 가능하다.

ProcessHandle.allProcesses()
    .forEach(p -> System.out.println(p.pid()));


리소스 사용량 모니터링

CPU와 메모리 사용량도 추적할 수 있다.

파이썬에서는 다음과 같이 가능하다.

import psutil

print(psutil.cpu_percent())
print(psutil.virtual_memory().percent)


이런 코드를 이용해
시스템 모니터링 프로그램이나 서버 상태 체크 도구를 만들 수 있다.

파일 시스템 감시

특정 폴더에 파일이 생성되거나 수정되는 것도 감지할 수 있다.

자바에서는 WatchService를 사용할 수 있다.

WatchService watchService = FileSystems.getDefault().newWatchService();


이걸 이용하면
로그 파일이 생성되는 순간 처리하거나
업로드 폴더를 감시하는 프로그램을 만들 수 있다.

프로세스 제어도 가능한가

추적뿐만 아니라 어느 정도 제어도 가능하다.

예를 들어
특정 프로그램을 실행시키거나
조건에 따라 종료시키는 것도 가능하다.

파이썬 예시

import subprocess

subprocess.run(["notepad.exe"])


이 코드는 메모장을 실행한다.

권한이 있다면
특정 프로세스를 종료하는 것도 가능하다.

다만 여기에는 중요한 제한이 있다.

무엇이 안 되는가

자바나 파이썬으로는
운영체제 자체의 내부 동작을 마음대로 바꿀 수는 없다.

커널 동작 변경
하드웨어 직접 제어
다른 사용자 영역 침범

이런 작업은 불가능하거나 매우 제한적이다.

이런 수준의 제어는
C나 C++ 같은 저수준 언어
또는 운영체제 커널 개발 영역이다.

그래서 제어 프로그램은 어디까지 쓰이나

현실적으로 자바나 파이썬으로 만들 수 있는 제어 프로그램은 다음과 같다.

시스템 모니터링 도구
서버 상태 감시 프로그램
로그 추적 프로그램
배치 작업 스케줄러
자동 실행 및 자동 종료 프로그램
파일 감시 및 처리 시스템
보안 이벤트 탐지 도구

이런 프로그램들은 실제 현업에서 많이 사용된다.

정리하면

컴퓨터에서 일어나는 많은 일들은
운영체제가 관리하고 있다.

자바와 파이썬은
운영체제가 제공하는 창구를 통해
그 일들을 충분히 추적하고 일부 제어할 수 있다.

즉

모든 것을 제어할 수는 없지만
실무에 필요한 대부분의 제어 프로그램은 구현 가능하다.

한 문장 요약

자바와 파이썬으로는
컴퓨터에서 일어나는 일을 관찰하고
필요한 수준까지 제어하는 프로그램을 만들 수 있다.

다만 그 한계는
운영체제가 허용한 범위까지다.
