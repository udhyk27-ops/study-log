# 클라우드 기초 개념

1. 클라우드란?
* 내 컴퓨터에서 서버 돌리는 것”의 한계
* 서버를 직접 사서 운영하는 방식(옛날 방식)
* 클라우드는 서버를 빌려 쓰는 개념
* 필요할 때 늘리고, 안 쓰면 줄이는 개념
* 왜 요즘 서비스는 대부분 클라우드를 쓰는지

2. IaaS / PaaS / SaaS를 비유로 이해해보기
* IaaS: 땅 + 집 뼈대만 빌리는 느낌
* PaaS: 집은 있고 가구만 배치
* SaaS: 호텔에 바로 들어가 쓰는 느낌
* 개발자가 신경 써야 할 것 차이

3. 클라우드에서 서버는 항상 켜져 있을까?
* 서버 = 항상 켜져 있어야 한다는 오해
* 필요할 때만 서버 사용 가능
* 트래픽이 많을 때/적을 때 차이
* 왜 “사용한 만큼만 비용”이 나오는지
* 간단한 예: 낮에만 사람 몰리는 서비스
