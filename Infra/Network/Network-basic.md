TCP와 UDP 그리고 네트워크 기본 개념
TCP와 UDP 차이
TCP

연결을 먼저 맺고 통신함

데이터 순서 보장

손실 시 재전송

신뢰성 높음

속도는 상대적으로 느림

주 사용처
웹 통신, 채팅, 파일 전송

UDP

연결 없이 바로 전송

데이터 순서 보장 안 됨

손실 가능

신뢰성 낮음

속도 빠름

주 사용처
게임, 스트리밍, 음성 통화

한 줄 요약

TCP는 정확함, UDP는 빠름

네트워크 기본 개념
IP 주소

네트워크에서 컴퓨터를 구분하는 주소

포트

한 컴퓨터 안에서 실행 중인 프로그램을 구분하는 번호

클라이언트와 서버

요청하는 쪽과 요청을 받는 쪽의 역할 구분

소켓

두 프로그램이 데이터를 주고받기 위한 통신의 끝점

프로토콜

데이터를 주고받기 위한 약속된 규칙

초간단 정리

네트워크 통신은 IP와 포트로 대상을 찾고,
TCP 또는 UDP 프로토콜을 통해 소켓으로 데이터를 주고받는다.
